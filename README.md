# Space Launch Tracker

A JavaScript web application that retrieves and displays upcoming space launch information using external APIs.

## Overview

This project demonstrates API integration and frontend rendering by displaying launch-related data in an organized and interactive web interface.

## Features

- Space launch tracking
- Dynamic API-driven content
- Interactive frontend updates
- Responsive page layout
- Real-time launch information display

## Technologies Used

- HTML5
- CSS3
- JavaScript
- REST APIs

## Concepts Demonstrated

- API integration
- Asynchronous JavaScript
- Frontend rendering
- Dynamic UI updates
- Responsive design principles

## What I Learned

Through this project, I gained experience with:
- Consuming real-world APIs
- Handling asynchronous data
- Updating frontend interfaces dynamically
- Structuring JavaScript applications
- Building responsive layouts
